# Data Cleaning Utility

## Objective
To clean raw datasets by handling missing values, fixing data types,
removing duplicates, and standardizing column names.

## Tools Used
- Python
- Pandas

## Tasks Performed
- Detected missing values
- Filled or removed null values
- Converted columns to correct data types
- Removed duplicate records
- Standardized column names
- Generated a cleaned dataset
- Created a cleaning log documenting changes

## Outcome
This project reflects real-world data preprocessing skills required in
professional data analysis roles.
