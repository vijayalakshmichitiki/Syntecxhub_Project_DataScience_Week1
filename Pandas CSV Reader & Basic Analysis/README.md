# Pandas CSV Reader & Basic Analysis

## Objective
To read structured data from CSV files and perform basic data analysis using
Pandas.

## Tools Used
- Python
- Pandas
- NumPy

## Tasks Performed
- Loaded CSV data into a Pandas DataFrame
- Inspected data using head, tail, and info
- Calculated summary statistics such as mean, min, max, and count
- Filtered rows and selected specific columns
- Saved filtered results to CSV and Excel files

## Outcome
This project showcases fundamental data analysis skills and the ability to
manipulate structured datasets using Pandas.
